Our ML Dialect

We are going to use OCaml, version 4.01.0. Earlier/later versions will work perfectly fine for your own coding, but the binary distibutions of the FbDK are very picky on which version they are run under; we will supply binaries for other versions if people are having problems getting 4.01.0 running.

OCaml is not completely compatible with the most standard dialect of ML, SML, but is nonetheless similar. It includes classes and objects, something not found in SML. We will often call OCaml just "Caml", you can view the two as synonyms.

Installing OCaml 4.01.0 on Your Machine

OCaml is free and you should install it on your computer.

  • For OCaml on Windows we recommend going to OCaml on Windows and installing the exe referenced there, ocaml-4.01.0-i686-mingw64-installer3.exe
  • For OCaml on Mac we recommend using the installer http://caml.inria.fr/pub/distrib/ocaml-4.01/ocaml-4.01.0-intel.dmg
  • For Linux (also works on Mac) we recommend you use OPAM, a highly flexible OCaml package manager. Start at OCaml.org's install page for instructions. The key steps are 1) install the opam binary via e.g. running the shell script opam_installer.sh, 2) run command opam switch 4.01.0 to build OCaml version 4.01.0; and 3) add the line eval `opam config env` to your .profile/.bashrc shell init file to point your shell to the ocaml binary that opam installed at ~/.opam/4.01.0/bin/ocaml. Note that the Debian/Fedora/etc install tools cannot be counted on for version 4.01.0.

OCaml is available on all department Linux machines via the shell command ocaml.

OCaml development environments

We strongly recommend you run OCaml from something besides the shell directly.
Eclipse OCaIDE
This is an Eclipse plugin for OCaml; it is the "official recommended method" and is what will be used in lecture. See information on using OCaIDE for more information.
Emacs
See information on running OCaml inside Emacs to get going.
vim OMLet
If you like vim, it now has a caml mode, OMLet. The above two modes are significantly better, all this mode offers is automatic indentation and keyword highlighting.
